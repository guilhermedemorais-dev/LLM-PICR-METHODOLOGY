# Firecrawl Ingest

Use Firecrawl as an ingestion layer, not as a mandatory step for every task.

## Use Cases

- scrape documentation websites
- capture reference pages for future skills
- extract structured knowledge for catalogs
- refresh outdated sources

## Output Targets

- new skill draft
- updated catalog entry
- MCP reference note
- Obsidian note

## Rule

Only ingest external content when the local catalog is insufficient or outdated.
