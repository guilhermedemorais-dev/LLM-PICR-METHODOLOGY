# PICR Vault Template

This folder is a minimal Obsidian vault starter for PICR.

Suggested folders:

- `catalogs/`
- `skills/`
- `mcps/`
- `memory/`
- `projects/`

Recommended flow:

1. copy this vault template
2. open it in Obsidian
3. add your real project context
4. link project notes to skills and MCPs
5. record successful routes in `memory/`
