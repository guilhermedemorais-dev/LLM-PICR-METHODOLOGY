# Firecrawl MCP

## Purpose

Acquire documentation and structured website content when the local catalog is missing or outdated.

## Tools Exposed

- crawl pages
- extract content
- refresh known sources

## Authentication

Requires Firecrawl credentials when running against hosted services.

## Use When

- local skills are insufficient
- official docs need to be captured into PICR assets
- you need source material for a new skill or MCP

## Avoid When

- the task is already covered locally
- the content is too dynamic to freeze into a skill
