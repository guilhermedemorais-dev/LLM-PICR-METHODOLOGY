# Obsidian MCP

## Purpose

Connect PICR assets to an Obsidian vault for navigation, linking, memory, and curation.

## Tools Exposed

- create notes
- update notes
- link skills and MCPs
- sync routing memory

## Authentication

Depends on the user's local Obsidian integration method.

## Use When

- the user wants a human-readable external brain
- skills, MCPs, and memory should become linked notes

## Avoid When

- the user only needs CLI-only operation
